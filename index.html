WebTube Features

Dislike Counter

Return YouTube Dislike restores the ability to see dislikes on YouTube. This extension aims to restore power to users by using a combination of archived like and dislike data, as well as the likes and dislikes made by extension users to show the most accurate ratings.

SponsorBlock

SponsorBlock is an extension that will skip over sponsored segments of YouTube videos. SponsorBlock is a crowdsourced browser extension that let's anyone submit the start and end time's of sponsored segments of YouTube videos. Once one person submits this information, everyone else with this extension will skip right over the sponsored segment.

Ad Blocker

WebTube blocks all ads on YouTube so that you can enjoy an Ad-Free experience.

Age Restriction Bypass

WebTube lets you watch age restricted videos with any account.

Clickbait blocker

WebTube intergrates a powerfull Clickbait Blocker that allows you to preview a video's content, comments,... before clicking on it, allowing you to avoid clickbait.
